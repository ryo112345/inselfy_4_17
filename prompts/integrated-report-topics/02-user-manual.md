### 第{{CHAPTER_NUM}}章: あなたの取扱説明書

**執筆指示（この章固有）:**

この章では、あたかも「上司や同僚に渡す取扱説明書」のように、この人の活かし方を第三者視点で書いてください。読者は本人ですが、書きぶりは「この人と一緒に働く人への説明」という体裁にしてください。

**構成（以下のサブセクションを太字見出しで区切る）:**

**この人のエンジンがかかる瞬間**
WVの重視価値観から、何を与えるとモチベーションが上がるかを2〜3個、具体的な仕事場面で描写。200文字以内。

**この人に合う仕事の渡し方**
CIのプロファイルから、どんなタスクの振り方・関わり方が合うかを説明。200文字以内。

**この人のNGパターン**
WVの非重視価値観から、やってはいけない関わり方を説明。「安定を売りにして引き止める」「細かく進捗を管理する」等。200文字以内。

**この人への効果的な声かけ**
上記を踏まえた、効果的な声かけフレーズを2〜3個、具体的なシチュエーションとともに提示。200文字以内。

**文体指示:**
- 実用的・客観的なトーン。マニュアルのような読みやすさ
- 各サブセクションは太字見出し＋短い説明文＋具体例の組み合わせ
- **必ずこの人の実際の職歴を踏まえた具体例を入れてください。**

**分析の切り口:**
- WV重視 → 「与えるべきもの」「エンジンをかけるスイッチ」
- WV非重視 → 「避けるべき関わり方」「やると逆効果なこと」
- CI高スコア領域 → 「任せるべき仕事の種類」
- CI低スコア領域 → 「無理にやらせないほうがいいこと」

**禁止事項:**
- 散文で長々と書かない（この章は構造化されたフォーマット）
- 抽象的な性格描写だけで終わらない（必ず具体的な仕事場面に落とす）
