## 注意事項

- スコアの具体的な数値（μ値、SE値、1〜5のスコア等）をレポート本文中に一切記載しないでください。数値ではなく、その人の人柄・情熱・行動傾向として表現してください。
- 「XXタイプです」というラベル貼りで終わらないでください。この人の実際の職歴に基づく具体的な記述をしてください。
- 具体的な統計データ、年収の数値、求人倍率などの数値情報は提示しないでください。
- 温かく前向きなトーンで、読み手を励ますように書いてください。ただし過度に褒めたり持ち上げたりせず、誠実さを保ってください。
- ネガティブな断定（「あなたには向いていません」）は禁止です。ズレの指摘は「〜かもしれません」と可能性として述べてください。
- レポート本文のみを出力してください。前置きや「以下はレポートです」のような導入文は不要です。
- ダッシュ記号（——、——、―、−、–、—）は一切使わないでください。補足や言い換えは「、」や句読点、括弧で表現してください。

## 思考プロセス

レポートを書き始める前に、まず以下のステップを踏んでください：

1. この人の職歴を時系列で読み、転職・異動のタイミングと理由を推測してください。
2. Work Values の重視/非重視パターンと、実際のキャリア選択の一致/不一致を分析してください。
3. Career Interest（RIASEC + 20基本興味）と、実際に就いてきた仕事内容の一致/不一致を分析してください。
4. 一致している部分と不一致の部分を整理してください。
