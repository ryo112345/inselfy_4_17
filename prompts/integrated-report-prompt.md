## レポート構成

以下の4章構成で、マークダウン形式で出力してください。
**重要: すべてのセクションで、1つの段落は最大200文字以内にしてください。200文字を超える場合は段落を分けてください。**

### 第1章: 面接・自己PRで使える「自分の言語化」

診断結果と経歴を突き合わせて、「なぜその選択をしてきたか」を一貫したストーリーとして表現した自己PR案を生成してください。
面接で「転職理由」や「キャリアの軸」を聞かれた時にそのまま使えるような、説得力のある文脈にしてください。
**必ずこの人の実際の会社名・職種・経歴に言及してください。**
2〜3段落、各200文字以内。

### 第2章: 3年後の自分への「問い」

結論やToDoリストではなく、この人が **考え続けるための問い** を2〜3個、提示してください。
診断結果と職歴から導かれた、この人固有の問いにしてください。汎用的な自己啓発的問いは禁止です。

### 第3章: 転職・異動の「判断パターン」分析

この人の過去の転職や異動のタイミングを、Work Values の診断結果と照合して分析してください。
「どんな状況になると動きたくなるのか」「何がトリガーになってきたか」というパターンを可視化してください。
**必ずこの人の実際の転職・異動の経歴に言及してください。** 汎用的な記述は禁止です。
2〜3段落、各200文字以内。

### 第4章: ユーザーからのリクエスト

ユーザーが自由記述で以下のリクエストを書きました:
「自分はこの道でいいのか他の道に行ったらいいのか示してほしい。」

このリクエストに対して、診断結果（Work Values + Career Interest）と職歴・スキルのデータを踏まえて、この人固有の回答を書いてください。
汎用的な回答は禁止です。必ずこの人のデータに基づいた具体的な内容にしてください。
2〜3段落、各200文字以内。

## 注意事項

- スコアの具体的な数値（μ値、SE値、1〜5のスコア等）をレポート本文中に一切記載しないでください。数値ではなく、その人の人柄・情熱・行動傾向として表現してください。
- 「XXタイプです」というラベル貼りで終わらないでください。この人の実際の職歴に基づく具体的な記述をしてください。
- 具体的な統計データ、年収の数値、求人倍率などの数値情報は提示しないでください。
- 温かく前向きなトーンで、読み手を励ますように書いてください。ただし過度に褒めたり持ち上げたりせず、誠実さを保ってください。
- ネガティブな断定（「あなたには向いていません」）は禁止です。ズレの指摘は「〜かもしれません」と可能性として述べてください。
- レポート本文のみを出力してください。前置きや「以下はレポートです」のような導入文は不要です。
- ダッシュ記号（——、——、―、−、–、—）は一切使わないでください。補足や言い換えは「、」や句読点、括弧で表現してください。

レポートを書き始める前に、まず以下のステップを踏んでください：

1. この人の職歴を時系列で読み、転職・異動のタイミングと理由を推測してください。
2. Work Values の重視/非重視パターンと、実際のキャリア選択の一致/不一致を分析してください。
3. Career Interest（RIASEC + 20基本興味）と、実際に就いてきた仕事内容の一致/不一致を分析してください。
4. 一致している部分と不一致の部分を整理してください。
